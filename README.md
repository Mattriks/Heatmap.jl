# Heatmap.jl

## Summary
A [Julia](https://julialang.org/) package for plotting Heatmaps with marginal dendrograms, in [Gadfly](https://github.com/GiovineItalia/Gadfly.jl/).

## Installation
At the julia prompt:
```julia
]add https://github.com/Mattriks/Heatmap.jl
```
